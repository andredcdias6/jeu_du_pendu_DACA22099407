# README pour le répertoire "jeu_du_pendu_DACA22099407"

Ce dépôt contient le code python pour jouer au jeu du pendu et le fichier mots_pendu.txt.

Auteur : André Da Costa Dias (DACA22099407)

Date et lieu : 23 mai 2023, Ecole de technologie de Montréal

## Description

Ce script permet à l'utilisateur de jouer en ligne de commande au jeu du pendu contre le programme. 
Ce dernier sélectionne aléatoirement un mot à partir du fichier texte "mots_pendu.txt"et guide l'utilisateur sur les actions à mener pour jouer.

## Contenu du dépôt

Le dépôt est organisé comme suit :

- `jeu_du_pendu.py` : Le script python pour exécuter le jeu.
- `mots_pendu.txt` : Ce fichier texte contient les mots à deviner par l'utilisateur que le script va piocher aléatoirement.

## Utilisation
Le fichier texte "mots_pendu.txt" doit obligatoirement être dans le même sous-dossier que ce script "jeu_du_pendu.py"). 

Pour exécuter le jeu du pendu, suivre les étapes :

1. Accéder au répertoire "jeu_du_pendu_DACA22099407" :
`cd jeu_du_pendu_DACA22099407`

2. Exécuter le script principal :
`python jeu_du_pendu.py`

3. Suivre les instructions affichées en ligne de commande pour interagir avec le script afin de jouer au Jeu du pendu.
